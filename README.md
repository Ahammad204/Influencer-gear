# Influencer Gear 🎬

A modern, professional e-commerce platform designed to showcase and sell high-quality equipment for content creators and influencers. This website provides a seamless shopping experience for photographers, videographers, and digital creators to discover and purchase premium production gear.

---

## 🌐 Live Demo

**Visit the live website:** [https://ahammad204.github.io/Influencer-gear/](https://ahammad204.github.io/Influencer-gear/)

## 📋 Project Overview

Influencer Gear is a responsive, single-page e-commerce website built with modern web technologies. It features an extensive collection of production equipment ranging from affordable accessories to professional-grade setups.

### Key Features

- **🎯 User-Friendly Navigation** - Intuitive menu with Home, Products, Sign Up, and Log In options
- **🛍️ Product Showcase** - Grid-based product catalog with 8+ premium equipment categories
- **⭐ Star Ratings** - Clear 5-star rating system for each product
- **🌍 Trusted Retailers** - Featured partnerships with major marketplaces (Amazon, Alibaba, eBay, Walmart, etc.)
- **📱 Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **🎨 Modern Design** - Professional color scheme with smooth animations and transitions
- **💳 E-Commerce Ready** - Shopping cart and checkout functionality

## 🔧 Customization

### Changing Colors
Edit the CSS variables in `styles/style.css`:
```css
:root{
    --btn-color: rgba(224, 44, 109, 1);      /* Primary button color */
    --text-color: rgba(58, 58, 58, 1);       /* Text color */
}
```

### Adding Products
Add new product cards to the `.card-container` in `index.html`:
```html
<div class="card">
    <img src="./images/your-product.png" alt="">
    <h1 class="card-title">Product Name</h1>
    <h4>$Price</h4>
    <!-- Rating stars -->
    <p class="card-details">Description</p>
</div>
```
